# Xonix-Arcade-Clone

A fast-paced, Xonix-inspired area-capture arcade game built from scratch in C++ using the SFML library. This project explores custom game engine architecture, recursive algorithms, and real-time AI logic.

## Key Features
* **Custom Engine:** Real-time game loop and tile-based grid management.
* **Intelligent AI:** Unique movement patterns (Circular, Zigzag, Linear) with adaptive difficulty scaling.
* **Area-Capture Logic:** Recursive flood-fill algorithm for territory detection and collision mechanics.
* **Dynamic Gameplay:** Tiered scoring with multipliers and a time-based "Enemy Freeze" power-up.
* **Persistence:** Local file-based storage for a Top 5 high-score leaderboard.
* **State Management:** Seamless transitions between Main Menu, Difficulty Select, and Gameplay.

## Technologies Used
* **Language:** C++
* **Graphics/Audio:** SFML (Simple and Fast Multimedia Library)
* **Logic:** Standard Template Library (STL), File I/O, Recursive Algorithms

## How to Run
1. **Clone the repo:** `git clone [Your-Repo-Link]`
2. **Install SFML:** Ensure [SFML](https://www.sfml-dev.org/) is installed on your system.
3. **Compile:** Build using your preferred C++ IDE or Makefile.
4. **Resources:** Keep the `images/` folder in the same directory as your executable to ensure textures and fonts load correctly.
